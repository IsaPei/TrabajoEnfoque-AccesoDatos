Proyecto en el que a partir de un archivo xml se introducen los datos a una base de datos,
a su vez se generará un nuevo documento xml con las restricciones oportunas desde la base de datos anterior.

El proyecto contiene una base de datos creada en local, PHPmyadmin. 
Para poder abrirlo es necesario el uso de XAMPP (activar Apache y SQL) e importar la base de datos en PHPmyadmin.
